# FlashcardsXBlock
An Open edX platform XBlock to create and display flashcards.

The flashcards are added as semicolon-separated values between the &lt;flashcards&gt; tags. Example:

```html
&lt;flashcards title="Capital cities"&gt;
Croatia;Zagreb
France;Paris
&lt;flashcards&gt;
```
