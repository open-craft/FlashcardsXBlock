# FlashcardsXBlock
An Open edX platform XBlock to create and display flashcards.

The flashcards are added as comma-separated values between the &lt;flashcards&gt; tags. Example:

>&lt;flashcards title="Capital cities"&gt;
>
>Croatia,Zagreb
>
>&lt;flashcards&gt;
